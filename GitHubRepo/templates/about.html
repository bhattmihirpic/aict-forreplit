About Academic Integrity Tool

A free, open-source solution for detecting plagiarism and AI-generated content in academic assignments.

Features
  • Plagiarism Detection: Advanced TF-IDF algorithms for text similarity analysis
  • AI Content Detection: Machine learning models to identify AI-generated text
  • Multi-format Support: PDF, DOCX, and TXT files
  • Detailed Reports: Comprehensive analysis with highlighted passages
  • Privacy Focused: All processing done locally, no external data sharing
  • Free Forever: No subscriptions, no limits, completely open source
Privacy & Security

Your documents are processed locally and automatically deleted after analysis. No content is stored permanently or shared with third parties. This tool respects student privacy while helping maintain academic integrity.

Open Source

This tool is built with open-source technologies and can be customized for your institution's needs. The source code is available for review, modification, and improvement.

Technologies Used
  • Python Flask (Web Framework)
  • scikit-learn (Machine Learning)
  • Transformers (AI Detection)
  • PyMuPDF (PDF Processing)
  • Bootstrap 5 (UI Framework)
  • SQLite (Database)
Best Practices
  • Use as educational tool, not punishment
  • Review results before making decisions
  • Consider context and proper citations
  • Combine with manual review
  • Maintain open dialogue with students
Created for Educators: This tool was designed to help teachers maintain academic integrity while supporting student learning. It's most effective when used as part of a comprehensive academic integrity program.
